# Cognisense
## Our project for the AI ATL 2024 Hackathon

<br />

Run the code

```
git clone https://github.com/leapingturtlefrog/AI-ATL-24.git
```

```
cd AI-ATL-24
```

```
terraform init
```

```
terraform apply
```

Take down the container

```
terraform destroy
```
